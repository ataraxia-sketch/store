ATARAXIA STORE V3 LIVECHAT

Isi folder:
- index.html = halaman toko pelanggan + widget livechat
- admin.html = dashboard admin v3
- assets/css/admin.css = style admin
- assets/css/livechat.css = style livechat pelanggan
- assets/js/firebase-config.js = config Firebase
- assets/js/admin.js = logika admin, produk, upload gambar, livechat
- assets/js/livechat-client.js = widget livechat pelanggan
- firebase/firestore_rules_v3.txt = rules Firestore terbaru
- firebase/storage_rules_v3.txt = rules Storage

Cara pakai:
1. Upload semua isi folder ke hosting.
2. Buka admin.html untuk login admin.
3. Publish Firestore rules v3 agar livechat bisa berjalan.
4. Storage rules tetap sama untuk upload gambar produk.
